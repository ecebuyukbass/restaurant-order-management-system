## Restaurant Order Management System

A backend-driven restaurant order management system built with Spring Boot, JPA, and H2 Database.

This project demonstrates layered architecture, entity relationships, RESTful API design, and order lifecycle management in a structured backend application.

## Features

Product Management,
Table-Based Order Creation,
Order Item Tracking,
Dynamic Total Price Calculation,
Order Status Lifecycle (Enum-based),
Admin Panel & Waiter Interface (Thymeleaf),
H2 In-Memory Database,
Basic Spring Security Configuration


## Tech Stack

Java 17,
Spring Boot,
Spring Data JPA,
Hibernate,
H2 Database,
Thymeleaf,
Maven

## How to Run
1️)Clone the repository
git clone https://github.com/ecebuyukbass/restaurant-order-management-system.git
2️)uild the project
mvn clean install
3)Run the application
mvn spring-boot:run

Application runs at:
http://localhost:8080

H2 Console:
http://localhost:8080/h2-console

Notes
H2 is used for development.
Designed for educational and architectural demonstration purposes.
